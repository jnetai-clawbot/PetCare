# PetCare 🐾

A comprehensive pet care coordinator app for Android.

## Features

- 🐶 **Pet Management** - Add pets with name, species, breed, date of birth, and photo
- 🍽️ **Feeding Schedules** - Set feeding times with reminders
- 🏥 **Vet Appointments** - Track appointments with 1-hour-before reminders
- 💊 **Medication Tracking** - Track medications with dose and frequency reminders
- ⚖️ **Weight Tracking** - Record weight with line chart visualization
- 💉 **Vaccination Records** - Track vaccinations with due date alerts
- 📝 **Health Notes** - Log notes per pet (General, Health, Behaviour, Diet)
- 📤 **Data Export** - Export pet data as JSON
- 🌙 **Dark Theme** - Material Design 3 dark theme
- ℹ️ **About Section** - Version info, update check, share

## Technical Stack

- Kotlin + AndroidX
- Room database
- Material Design 3 (dark theme)
- AlarmManager for reminders
- Coroutines (Dispatchers.IO for all DB operations)
- MPAndroidChart for weight graphs

## Building

1. Clone the repo
2. Open in Android Studio
3. Build & run
